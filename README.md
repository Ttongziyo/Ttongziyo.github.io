# 小猫补光灯应用

一个基于网页的实时补光灯应用，通过屏幕反光实现面部补光效果。

## 功能特点

- 实时摄像头预览
- 可调节背景颜色
- 多种滤镜效果
- 亮度、饱和度、色相调节
- 可爱少女风格界面设计

## 使用方法

1. 打开 `index.html` 文件
2. 允许浏览器访问摄像头
3. 使用控制面板调节补光效果

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)
- WebRTC (摄像头访问)

## 注意事项

- 需要现代浏览器支持
- 需要允许摄像头访问权限
- 建议在光线充足的环境下使用
